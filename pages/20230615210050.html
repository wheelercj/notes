C++ command line arguments

Sometimes it's helpful to create your own terminal commands similar to this add command I made. This can be done as explained in [ยง] how to create custom terminal commands, but giving inputs (like the two numbers above) to the custom command is another matter.

Every widely-used general-purpose programming language has a way to get command line arguments. Here's an example in C++:

#include <iostream>
using namespace std;

int main(int argc, char* argv[])
{
    int sum = 0;
    for (int i = 1; i < argc; i++)
    {
        sum += atoi(argv[i]);
        if (i < argc - 1)
            cout << argv[i] << " + ";
    }
    cout << argv[argc - 1] << " = " << sum << endl;
    return 0;
}

The unusual part is that the main function is taking inputs, int argc, char* argv[]. The parameter names argc and argv can be almost anything you want, but their types and order are important. argv (or whichever name you choose) is an array of c-strings of all the inputs to the program (the argument values). argc is the size of the argv array (the argument count). The first element of argv is always the path to the program.

After setting up a custom terminal command for this (example below), using the terminal command add 267 917 puts the value "267" in argv[1] and "917" in argv[2]. As shown above, these c-strings of numbers can then be converted to ints with the atoi function and added together.
