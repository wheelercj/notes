how to publish a zettelkasten

Zettelkastens are usually made out of plaintext files, which makes using a static site generator (SSG) perfect for publishing a zettelkasten online. The most popular SSGs (such as Jekyll, Hugo, and Gatsby) can be difficult to use with the zettelkasten format, but others have been specifically made for them. A few of these are listed below. I built one of them and haven't tried the others, but they all look promising in different ways.

  • Emanote has a lot of work put into it.
  • Writer's Block looks versatile, but may require more setup time than the other options here.
  • zk-ssg focuses on minimizing time between writing and publishing, but currently only has the essential features (this is the one I made).
  • Zettel-Hugo-Postmaker can be good if you want to use Hugo. It can change some internal link formats into Hugo's format.
  • Obsidian has a feature that allows users to publish files online for $20 per month per site (this isn't actually an SSG, but it works with some zettelkastens).
