swapping pointers in C++

Swapping pointers is a little different than swapping normal variables. There are two ways to do this.

Assuming a class called Book has been defined, and an array of type Book called books has been created . . .

You can swap the pointers themselves:

void swap(Book** book1, Book** book2)
{
    Book* temp = *book1;
    *book1 = *book2;
    *book2 = temp;
}

Then the function call can look something like this:

swap(&books[i], &books[j]);

Or instead of swapping the pointers themselves,
you can swap what the pointers are pointing to:

void swap(Book& book1, Book& book2)
{
    Book temp = book1;
    book1 = book2;
    book2 = temp;
}

Here's the function call:

swap(*books[i], *books[j]);

To compare,
Sometimes it's better to swap the pointers themselves because then less information has to be moved. Sometimes it doesn't make much of a difference which method is used.
