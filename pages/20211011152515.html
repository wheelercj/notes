swapping pointers in C++

If you want to swap two pointers or what two pointers are pointing to, you have (at least) three options for how to do that.

#include <iostream>
using namespace std;

void swap_values(int& num1, int& num2)
{
    int temp = num1;
    num1 = num2;
    num2 = temp;
}

void swap_pointers1(int** num1, int** num2)
{
    int* temp = *num1;
    *num1 = *num2;
    *num2 = temp;
}

void swap_pointers2(int*& num1, int*& num2)
{
    int* temp = num1;
    num1 = num2;
    num2 = temp;
}

void print(int* num1, int* num2)
{
    cout << *num1 << " " << *num2 << endl;
}

int main()
{
    int* num1 = new int(2);
    int* num2 = new int(1);
    print(num1, num2);  // 2 1
    swap_values(*num1, *num2);
    print(num1, num2);  // 1 2
    swap_pointers1(&num1, &num2);
    print(num1, num2);  // 2 1
    swap_pointers2(num1, num2);
    print(num1, num2);  // 1 2

    delete num1;
    num1 = NULL;
    delete num2;
    num2 = NULL;
    return 0;
}

The last two options are more efficient because the total amount of data copied while the functions are running is only a few memory addresses, even if they point to variables taking up massive amounts of memory. The most significant syntax difference is in the function calls; swap_pointers2's arguments don't need any special characters next to them just for the function to work.

Of course, all of these swap functions can be used on array elements, such as swap_pointers2(numbers[3], numbers[4]);, where numbers is an array of pointers.
