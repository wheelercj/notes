how to create custom terminal commands (Windows)

With terminal apps such as Windows Terminal, you can create your own custom terminal command to make running scripts and apps easier.

  1. Create a .bat file named after what you want the terminal command to be. For example, if you want to create a todo command that adds tasks to a to do list by running a Python script, create todo.bat.
  2. Put into this new file the terminal command(s) that will run the script/app. Continuing with the above example, the contents of the file could be py -3.10 "C:\Users\chris\Documents\programming\refactor todo list\create_task.py" "%*".
  3. Add the path to your new .bat file to the Path user environment variable.
    1. Copy the file's path.
    2. Press the Windows key.
    3. Type path and press Enter.
    4. Click Environment Variables.
    5. In the top half of the window, select the Path variable and click Edit.
    6. Click New and paste the path.

Now you can run your script by just opening Windows Terminal and typing the command you created without worrying what the current working directory is.
