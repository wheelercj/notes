how to create custom terminal commands

Creating custom terminal commands can change having to type something long like this:

py -3.10 "C:\Users\chris\Documents\programming\refactor todo list\create_task.py"

to something like this:

todo

And it will work no matter which directory the terminal is in. How to create custom commands depends on your operating system.

Mac and Linux

This guide seems well-written (no need to rewrite the wheel).

Windows

I recommend checking out Windows Terminal if you're choosing a terminal app. Here's how to create custom commands:

  1. Create a .bat file named after what you want the terminal command to be. For example, if you want to create a todo command that adds tasks to a to do list by running a Python script (like this one), create todo.bat.
  2. Put into this new file the terminal command(s) that will run the script/app. Continuing with the above example, the contents of the file could be py -3.10 "C:\Users\chris\Documents\programming\refactor todo list\create_task.py" "%*".
  3. Add the path to your new .bat file (or the folder it's in) to the Path user environment variable.
    1. Copy the file's or folder's path.
    2. Press the Windows key.
    3. Type path and press Enter.
    4. Click Environment Variables.
    5. In the top half of the window, select the Path variable and click Edit.
    6. Click New and paste the path.

Now you can run your script by just opening your terminal and typing the command you created without worrying what the current working directory is.

If you will create multiple custom terminal commands, you could put all the .bat files into a folder and only add the folder to your Path environment variable.
