short list of software recommendations

Here's software that I use often, highly recommend, have no affiliation with, and doesn't seem to be super well known or understood.

  • Use a password manager. Choices related to security are usually a trade-off between convenience and security, but password managers increase both in the long run. I happen to use Bitwarden and it's great; it has an excellent free tier and can optionally be self-hosted. I've also heard good things about 1password and KeePass. I recommend avoiding password managers built into a specific operating system or browser in case you want your passwords to be available elsewhere in the future.
  • Depend on search features. I sometimes see people spend a lot of time deciding which folder to put a file/bookmark/etc. into, and also a lot of time searching through their folders for something. Many applications have great search features that allow you to spend little to no time organizing but still quickly find what you need. Find these search features and make use of them to save tons of time. Some applications have significantly better search features than others; consider this when choosing between apps.
  • Separate how you keep track of events and tasks. If you use apps, use one app that specializes in being the best calendar, and another app that specializes in being the best to do list. Trying to combine them will slow you down.
  • Run automated backups. This is different from syncing. If you only sync your files, the accidental loss of important files will also sync. But with good backup software, you will have multiple copies of all your important files, each from a different time. Having automated backups can also give you the freedom to take more risks and save time. Duplicati backing up to Backblaze B2 Cloud Storage has been working well for me, and there are other options out there that are easier to use but have fewer features or cost more. At the time of writing, I'm paying about $0.02 USD per month to back up about 33 GB.
  • Back up your emails. If you use Gmail or a similar email service, you can set up automated backing up of your emails by connecting it to any local email client (such as Thunderbird). If you're interested, I recommend searching online for a guide for exactly how to do this (you may need to change an email setting).
  • For Windows users: PowerToys, a collection of extremely helpful tools made by Microsoft, such as for resizing images, bulk file renaming, creating custom keyboard shortcuts, changing what your mouse buttons do, window management, and much more.
