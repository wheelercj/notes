intro to version control (with git and GitHub)

Here's a great way to learn how to use git and GitHub:

  1. Watch these short videos: Version control with Git and GitHub by freeCodeCamp.
  2. Use Sublime Merge, a git client with a graphical interface, because it makes the most commonly used features of git easier to use without preventing you from learning about git.
  3. Read some of this free online book, but only the parts that are relevant to you: Pro Git by Scott Chacon and Ben Straub.

specific topics

  • If you're working on a big project, read A successful Git branching model by Vincent Driessen, which is an article about workflows with git (sometimes called "git-flows").
  • If you want to use version numbers for your project, there are multiple options; semantic versioning is covered in Versioning by datasift.
