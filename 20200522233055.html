practice: printing shapes with loops

Write a program that prints out the following shapes using only a single asterisk in the code for each shape (not including any multiplication). You may use as many loops and variables as you want as long as you don't assign an asterisk to a variable. The shapes start easy and gradually get more difficult, with the last few being very difficult especially if you don't have some experience with graphing and algebra. I recommend keeping the code for each shape as you go.
